CCTS NDR Exporter

The CCTS NDR Exporter provides functionality to export UPCC document models according to the UN/CEFACT XML Naming and Design Rules, Version 3.0, ODP6.

In addition to exporting document and business libraries, the exporter also supports exporting core component libraries.

The following UPCC library types can be exported:

  • BDT Library
  • BIE Library
  • DOC Library
  • CDT Library
  • CC Library

ENUM Libraries are currently not supported.
PRIM Libraries are not exported, because PRIMs map directly to XSD basic types.
